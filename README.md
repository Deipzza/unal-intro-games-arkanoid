# Especificaciones.
* Los power up implementados son:
    * **Speed:** aumento y disminución de la velocidad de la bola.
    * **Scale:** aumento, disminución y restablecimiento del tamaño del paddle.
    * **Score:** sumar 50, 100, 250 y 500 puntos al jugador.
